# Teakozi Viewer

This app is a companion to Teakozi testing framework

  - Shows the HTML page report from the Logs generated by Teakozi (v1.1.x)


# How to install

 Clone the repository, get the dependencies, and run

```sh
$ git clone <this repo>
$ npm install
$ npm start
```
Access via http://localhost:3070
